# Team Management Application

A Laravel 12 application for managing teams and brokers.

## Requirements

- Docker
- Docker Compose

## Installation

```bash
./setup.sh
```

The application will be available at http://localhost:8000

## Default Credentials

- Email: admin@example.com
- Password: password

## Features

- View all teams
- View team details with brokers
- Add brokers to a team
- Delete a team (soft delete)
